# hackintosh
My hackintosh EFI (ASUS X299 PRIME DELUXE II + i9 9900X + Radeon VII)

### BIOS Setup

- MSR lock [disabled]
- XMP [enabled, profile#1]
- CSM [disabled]
- Fast boot [disabled]
- On-board bluetooth [disabled]
